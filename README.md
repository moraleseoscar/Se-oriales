# Nachos Landing

Landing page de Senorial Nachos construida con React, TypeScript y Vite.

## Requisitos
- Node.js (LTS)
- npm

## Desarrollo
```bash
npm install
npm run dev
```

## Build y preview
```bash
npm run build
npm run preview
```

## Scripts utiles
- `npm run dev` inicia el servidor de desarrollo
- `npm run build` compila TypeScript y genera la build
- `npm run preview` sirve la build localmente
- `npm run lint` ejecuta ESLint

## Estructura principal
- `src/App.tsx` contenido y secciones de la landing
- `src/styles.css` estilos globales
- `public/assets` imagenes y recursos

## Personalizacion rapida
- Cambia textos y secciones en `src/App.tsx`
- Ajusta estilos en `src/styles.css`
- Reemplaza imagenes en `public/assets`
